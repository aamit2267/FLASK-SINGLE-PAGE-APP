This is an single page portfolio made using `flask`.

##### Setup

```sh
$ git clone https://github.com/aamit2267/FLASK-SINGLE-PAGE-APP.git
$ cd ./FLASK-SINGLE-PAGE-APP/
$ pip install flask
```

##### Test

```sh
$ export FLASK_APP=app.py
$ flask run
```
